**Personal Budget Tracking Application**

 **Project Description**
The Personal Budget Tracking Application is a full-stack web application designed to help users efficiently manage their personal finances.  
It enables users to record income and expenses, allocate budgets for different categories, and analyze spending through visual reports.
This project was developed as part of my Internship Program to demonstrate full-stack development skills using Angular and Django.

Technology Stack
**Frontend**
	Angular
	TypeScript
	HTML5
	CSS3
	Bootstrap
**Backend**
	Django
	Django REST Framework
**Database**
	SQLite
Tools & Platforms
	Git & GitHub
	Visual Studio Code
	PyCharm
	Node.js & npm

**Key Features**
	Secure User Authentication
	Add / Update Income and Expenses
	Category-wise Budget Allocation
	Recent Activity Tracking
	Graphical Reports (Charts)
	Responsive User Interface

**System Architecture**
o	Angular frontend communicates with Django backend via REST APIs
o	Django processes requests and interacts with the database
o	JSON responses are sent back to the frontend for display
